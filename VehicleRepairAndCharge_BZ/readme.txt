PRAWN SUIT REPAIR AND CHARGE BZ - 1.0
Fully charge and repair the Prawn Suit when docking in the Moonpool or SeaTruck Docking module.

:: REQUIREMENTS ::
* No consideration for the legality of modifying Alterra firmware

:: FEATURES ::
* Configurable through the Mod menu in game
